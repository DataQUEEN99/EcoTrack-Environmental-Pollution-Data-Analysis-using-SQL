# EcoTrack-Environmental-Pollution-Data-Analysis-using-SQL
EcoTrack is a data analysis project built in **Google Colab** using **SQL (SQLite)** and **Python** to study air and water pollution trends across different cities.   It analyzes key air quality indicators — **PM2.5, PM10, SO₂, NO₂, and CO** — along with water quality and industrial emissions data
